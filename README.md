# AIP: Tic Tac Toe AI Game

This project was developed for the **Artificial Intelligence Principles (AIP)** module. It demonstrates the implementation of AI algorithms in a classic Tic Tac Toe game using Python and Pygame.

---

## 📌 Project Overview

This project is an interactive Tic Tac Toe game where a human player competes against an AI opponent with different difficulty levels.

The system applies artificial intelligence techniques such as decision-making, rule-based logic, and the Minimax algorithm to simulate intelligent gameplay.

---

## 🤖 AI Features

The game includes three levels of AI difficulty:

### 🟢 Easy Mode

* AI selects moves randomly
* No strategy applied

### 🟡 Medium Mode

* AI checks for:

  * Winning moves
  * Blocking player moves
* Basic decision-making logic

### 🔴 Hard Mode

* Uses **Minimax Algorithm**
* Evaluates all possible moves
* Always chooses the optimal move
* Nearly impossible to beat
